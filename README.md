## Tech stack

- [Next.js](https://nextjs.org) 16 (App Router)
- [React](https://react.dev) 19
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com) 4

## Prerequisites

- Node.js 20+
- npm (or yarn/pnpm/bun)

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app. Edit `app/page.tsx` — the page auto-updates as you save.

## Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Start the development server |
| `npm run build` | Build the app for production |
| `npm run start` | Run the production build |
| `npm run lint` | Lint the codebase with ESLint |

## Project structure

```
app/
  layout.tsx    # Root layout
  page.tsx      # Home page
  globals.css   # Global styles (Tailwind)
public/         # Static assets
```

## Deployment

Deploy on [Vercel](https://vercel.com/new) or any platform that supports Next.js. See the [deployment docs](https://nextjs.org/docs/app/building-your-application/deploying) for details.
