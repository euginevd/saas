## 📖 What is this?

This repository is the starting point for a SaaS product, currently scaffolded with Next.js, React, TypeScript, and Tailwind CSS. It exists to give the project a clean, modern foundation to build on — routing, styling, and tooling are already wired up so feature work can start without setup overhead. There is no shipped product yet; this README documents the project as it stands and will expand as real features land.

## 🌳 Repository structure

```
saas/
├── app/
│   ├── layout.tsx     # Root layout
│   ├── page.tsx       # Home page
│   ├── globals.css    # Global styles (Tailwind)
│   └── favicon.ico
├── public/            # Static assets
├── next.config.ts     # Next.js configuration
├── tsconfig.json      # TypeScript configuration
└── package.json       # Dependencies and scripts
```

## 🚀 Quick start

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
